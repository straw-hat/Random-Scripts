/** Random Scripts Read Me File **/


ImageChecker 
- for checking if images have proper naming as per xml data
- the file should be run from parent folder of 'Logo' folder
- it takes input from in and generates ImageData Content to OutData
- Also change the gamename in paths (*Imp)
- It prints errors to console
- Copy the OutData to ImageData only if no errors are present

XmlChecker
- Checks XML answer data
- Replaces XML options data (*Imp)
- Change the gamename in paths (*Imp)
- On a side effect prints extralines in xml file
- To remove extra lines open xml in notepad++ and replace '\r\r' with '\r'


PNGConverter
- Renames JPG to PNG in a folder
- Modify path accordingly
